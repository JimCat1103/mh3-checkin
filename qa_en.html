FAQ / Q&A

Q1. How to complete check-in and sign-back?

Enter your hash name and phone number to check in. After successful check-in, you'll see an "I'm Back" button. Click this button after the run to complete sign-back.

Q2. Can I check in for others?

Yes, you can add buddies on the check-in page. The buddy's phone can be left blank (will use the primary contact's phone), or if filled in, the buddy can later use their own phone to load their data and complete sign-back independently.
- If already signed back: System will show the completed page without modifying data.
- If not signed back: Shows "Already Checked In" screen, with the primary contact still displayed for identification.

Q3. How to modify Bash registration?

Before sign-back: Check in again with the same phone to update Bash status.
After sign-back: You can select Bash when clicking sign-back; if already signed back, please inform the organizer on-site.

Q4. Re-enter or check in for others?

If not yet signed back, click "Check-in for Others / Re-enter" on the check-in success page to re-enter. Your old data will remain in the system; you can check in again or query using the original phone number.

Q5. Event opening times and completion?

Registration opens automatically based on the weekly run. When not open, it will show the next run info. After sign-back completion, you can still view the list, but signed-back data won't be modified.

Q6. How to view the current check-in list?

Click "View Check-in Status" on the home page, check-in success page, or sign-back completion page. If you have checked in, your or your team's data will be prioritized at the top for easy viewing.
